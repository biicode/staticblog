As you report, we experience the same problem compiling the libraries for this project. The block lasote/carriots_stream is working fine but having troubles with lasote/weather.

It seems to be a problem with the AVR compiler inside the Arduino SDK. We use an AVR’s version distributed with Arduino’s 1.0.6 version.

A possible solution would be using the latest AVR GCC following this “guide”: http://forum.arduino.cc/index.php/topic,37965.msg281176.html#msg281176

It is a well-known error with the AVR linker: http://stackoverflow.com/questions/8188849/avr-linker-error-relocation-truncated-to-fit

We are still working on it and will update to the most recent version of AVR and Arduino as soon as we get a stable version of the arduino SDK.

Hope this helps you and keep trying more arduino examples!
