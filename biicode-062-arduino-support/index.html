Biicode 0.6.2 with Arduino Support

Posted on January 28, 2014 by julia

Hello bees!

We just released biicode 0.6.2 containing a few major features you can check here but our battleship for this release is Arduino support!

Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It’s intended for artists, designers, hobbyists and anyone interested in creating interactive objects or environments you can learn more about it in their official webpage.

Biicode provides you with several advantages for your Arduino experience.

  • Save your project in different folders for a better organization.

  • Reuse yours and other users code. Reusing made easy, just #include the file you need to get it. No more copy & paste files from project to project.

  • It is a very simple and fast way to compile and upload your code to your Arduino.

Start hacking with your Arduino here!
