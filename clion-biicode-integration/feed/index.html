]]> By: Maríahttp://blog.biicode.com/clion-biicode-integration/#comment-214 Thu, 26 Mar 2015 15:12:00 +0000 http://blog.biicode.com/?p=2144#comment-214 Hi Marius,

You have to tell CLion where to find those headers.

As far as I’ve used CLion the way to do so is setting include_directories variable in CMakeLists.txt to a proper value and CLion will search the headers following these paths.

INCLUDE_DIRECTORIES(/path/to/sdk/goes/here)

If you installed Arduino SDK along with biicode paths are:

MAC: ~/.biicode_env/arduino-1.0.6/Arduino.app/Contents/Resources/Java/hardware/arduino/cores/arduino
Linux: ~/.biicode_env/arduino-1.0.6/hardware/arduino/cores/arduino
Windows: C:biicode_envarduino-1.0.6hardwarearduinocoresarduino

We’re available at our forum (forum.biicode.com) for any further troubleshooting. Let me know if you work this out!
